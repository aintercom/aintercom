🔄 Forcer le rafraîchissement de AINTERCOM PRO

Les modifications ont été déployées avec succès sur https://live.aintercom.fr/app.html, mais votre navigateur utilise une version mise en cache.

📱 Méthode 1 : Rafraîchissement forcé (recommandé)

Sur Mac : Appuyez sur Cmd + Shift + R

Sur Windows/Linux : Appuyez sur Ctrl + Shift + R (ou Ctrl + F5)

🔧 Méthode 2 : Outils de développement

1. Ouvrez les outils de développement (F12)

2. Allez dans l'onglet "Network"

3. Cochez "Disable cache"

4. Rafraîchissez la page (F5)
